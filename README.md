## Getting started

1. Install [Node.js](https://www.npmjs.com/get-npm).

   `$ brew install node`

2. Clone this repo and cd into the directory.
3. Install all the dependencies.

   `$ yarn install`

4. Start your server.

   `$ yarn start`

5. Open a new tab and navigate to `http://localhost:3000` to see your app.
